# 员工管理系统 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目名称
员工管理系统 (Employee Management System, EMS)

### 1.2 项目目标
构建一个现代化的员工管理系统，支持员工信息的录入、管理、查询和导出，提升企业人事管理效率。

### 1.3 目标用户
- HR 管理人员
- 部门主管
- 系统管理员

---

## 2. 功能需求

### 2.1 员工信息字段定义

| 字段名 | 字段标识 | 类型 | 必填 | 说明 |
|--------|---------|------|------|------|
| 姓名 | name | string | 是 | 员工真实姓名 |
| 工号 | employeeId | string | 是 | 系统自动生成，格式: EMP-YYYYMMDD-XXX |
| 部门 | department | enum | 是 | 下拉选择：技术部/产品部/市场部/财务部/人事部/运营部 |
| 职位 | position | string | 是 | 员工职位名称 |
| 手机号 | phone | string | 是 | 11位手机号 |
| 邮箱 | email | string | 是 | 工作邮箱 |
| 入职日期 | hireDate | date | 是 | 入职时间 |
| 状态 | status | enum | 是 | 在职/离职/试用期 |
| 性别 | gender | enum | 否 | 男/女 |
| 生日 | birthday | date | 否 | 出生日期 |
| 地址 | address | string | 否 | 居住地址 |
| 备注 | remark | string | 否 | 附加信息 |

### 2.2 表格录入功能

- 支持 Excel/CSV 文件批量导入员工数据
- 导入前预览数据，显示校验结果
- 支持字段映射（文件列名 → 系统字段）
- 导入错误行高亮提示，支持修正后重新导入
- 提供导入模板下载

### 2.3 手动创建员工

- 表单页面，包含所有员工信息字段
- 实时表单校验（手机号格式、邮箱格式、必填项）
- 工号自动生成
- 创建成功后跳转至员工详情页

### 2.4 员工管理功能

#### 查询/筛选
- 关键词搜索（姓名、工号、手机号）
- 按部门筛选
- 按状态筛选（在职/离职/试用期）
- 按入职日期范围筛选
- 分页展示，支持调整每页条数

#### 编辑
- 点击员工记录进入编辑页面
- 支持修改所有可编辑字段（工号不可修改）
- 编辑历史记录

#### 删除
- 单条删除（二次确认）
- 批量删除（勾选后批量操作）
- 软删除机制，数据可恢复

#### 导出
- 导出当前筛选结果为 Excel/CSV
- 支持选择导出字段
- 导出全部数据

---

## 3. 非功能需求

- 响应式设计，适配桌面端和平板
- 页面加载时间 < 2s
- 支持 1000+ 员工数据流畅操作
- 数据本地持久化（localStorage/IndexedDB）
