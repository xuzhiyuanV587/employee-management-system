# 员工管理系统 - 用户操作指南

## 快速开始

### 启动系统
```bash
# 启动后端服务
cd server && npm start    # 运行在 http://localhost:3001

# 启动前端应用
cd frontend && npm run dev  # 运行在 http://localhost:3000
```

浏览器访问 `http://localhost:3000` 即可使用系统。

---

## 功能操作说明

### 1. 员工列表

进入系统后默认显示员工列表页，包含以下功能：

**搜索与筛选**
- 顶部搜索框：输入姓名、工号或手机号进行模糊搜索
- 部门筛选：下拉选择部门（技术部/产品部/市场部/财务部/人事部/运营部）
- 状态筛选：选择在职/离职/试用期
- 入职日期：选择起止日期范围
- 点击"搜索"执行查询，点击"重置"清空条件

**分页**
- 底部分页栏支持切换页码
- 可调整每页显示条数

**批量操作**
- 勾选表格左侧复选框选中员工
- 点击"批量删除"删除选中记录（需二次确认）

---

### 2. 创建员工

点击列表页"新增员工"按钮进入创建页面。

**必填字段：**
- 姓名、部门（下拉选择）、职位、手机号、邮箱、入职日期、状态

**选填字段：**
- 性别、生日、地址、备注

**注意事项：**
- 工号由系统自动生成，格式为 `EMP-YYYYMMDD-XXX`
- 手机号必须为 1 开头的 11 位数字
- 邮箱必须为有效格式
- 填写完成后点击"提交"，成功后自动跳转至员工详情页

---

### 3. 编辑员工

在员工列表中点击"编辑"按钮，进入编辑页面。

- 工号为只读，不可修改
- 其他字段均可修改
- 修改后点击"保存"提交

---

### 4. 查看员工详情

在员工列表中点击"查看"按钮，进入详情页。

- 展示员工所有信息
- 可从详情页直接跳转编辑或返回列表

---

### 5. 删除员工

**单条删除：** 点击列表中的"删除"按钮，确认后删除
**批量删除：** 勾选多条记录，点击"批量删除"

> 系统采用软删除机制，数据不会被物理删除。

---

### 6. 数据导入

点击列表页"导入"按钮，打开导入对话框。

**操作步骤：**
1. 下载导入模板（点击"下载模板"）
2. 按模板格式填写员工数据
3. 上传 Excel（.xlsx）或 CSV 文件（支持拖拽上传）
4. 预览导入数据，检查校验结果
5. 错误行会高亮提示，修正后可重新上传
6. 确认无误后点击"导入"

---

### 7. 数据导出

点击列表页"导出"按钮，打开导出对话框。

- 可选择需要导出的字段
- 导出范围为当前筛选条件下的数据
- 支持 Excel 和 CSV 格式
- 点击"导出"后自动下载文件

---

## 常见问题

**Q: 工号可以自定义吗？**
A: 不可以。工号由系统按 `EMP-日期-序号` 格式自动生成，确保唯一性。

**Q: 删除的员工数据能恢复吗？**
A: 系统使用软删除，数据保留在数据库中，可由管理员恢复。

**Q: 导入文件有大小限制吗？**
A: CSV 文件上传限制为 5MB。建议单次导入不超过 1000 条。

**Q: 支持哪些浏览器？**
A: 推荐使用 Chrome、Firefox、Edge 等现代浏览器的最新版本。

**Q: 手机号格式要求是什么？**
A: 必须为 1 开头的 11 位中国大陆手机号。

**Q: 如何修改部门列表？**
A: 当前版本部门为固定选项（技术部/产品部/市场部/财务部/人事部/运营部），如需调整请联系管理员。
