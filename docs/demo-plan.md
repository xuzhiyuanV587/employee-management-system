# 员工管理系统 - 项目演示方案

## 演示概要

- 演示时长：15-20 分钟
- 演示环境：本地开发环境（前端 :3000 + 后端 :3001）
- 演示人员：产品经理主讲，前后端开发支持

---

## 演示流程

### 第一幕：系统概览（2分钟）
1. 打开系统首页，展示员工列表空状态
2. 简要介绍系统功能模块和技术架构

### 第二幕：创建员工（3分钟）
1. 点击"新增员工"，展示完整表单
2. 演示表单校验 — 故意留空必填字段、输入错误手机号
3. 正确填写信息，提交创建
4. 展示自动生成的工号
5. 查看员工详情页

### 第三幕：批量导入（3分钟）
1. 下载导入模板
2. 上传预先准备好的测试 Excel 文件（10条数据）
3. 展示数据预览和校验结果
4. 确认导入，返回列表查看结果

### 第四幕：查询与筛选（3分钟）
1. 关键词搜索 — 输入"张三"
2. 部门筛选 — 选择"技术部"
3. 状态筛选 — 选择"在职"
4. 组合筛选演示
5. 分页切换

### 第五幕：编辑与删除（3分钟）
1. 编辑一条员工记录，修改职位和状态
2. 展示工号不可编辑
3. 单条删除（展示确认弹窗）
4. 批量勾选 + 批量删除

### 第六幕：数据导出（2分钟）
1. 设置筛选条件
2. 选择导出字段
3. 导出 Excel 文件
4. 打开导出文件验证数据正确性

### 第七幕：Q&A（3-5分钟）
- 回答观众提问
- 收集反馈意见

---

## 演示数据准备

使用 `docs/test-cases.md` 中的 10 条示例数据，覆盖：
- 6 个部门
- 3 种状态（在职/离职/试用期）
- 男女比例均衡
- 不同入职时间段

## 演示前检查清单

- [ ] 后端服务正常启动（port 3001）
- [ ] 前端应用正常启动（port 3000）
- [ ] 数据库已清空或重置
- [ ] 导入模板 Excel 文件已准备
- [ ] 浏览器缓存已清除
- [ ] 网络连接正常
